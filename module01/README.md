# Module 01 - AI Dev Tools Zoomcamp

This module contains homework assignments related to AI-assisted development.

## Contents

### homework1/ - Django TODO Application
A complete Django web application built with AI assistance for managing TODO items.

**Quick Start:**
```bash
cd homework1
source ../../.venv/bin/activate
python manage.py runserver
```

Visit: http://127.0.0.1:8000/

**Documentation:**
- [HOMEWORK_ANSWERS.md](homework1/HOMEWORK_ANSWERS.md) - Answers to all homework questions
- [PROJECT_OVERVIEW.md](homework1/PROJECT_OVERVIEW.md) - Complete project documentation

**Features:**
- Create, edit, and delete TODOs
- Assign due dates to tasks
- Mark TODOs as resolved/unresolved
- Fully tested (11 passing tests)
- Clean, responsive UI

**Technology Stack:**
- Django 5.2.8
- Python 3.13.9
- SQLite database
- Installed with `uv` package manager

**Run Tests:**
```bash
cd homework1
source ../../.venv/bin/activate
python manage.py test
```

All 11 tests pass successfully!
